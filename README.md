\# 📌 О проекте



\*\*YouTube Monster Bot\*\* — это многофункциональный Telegram-бот нового поколения, созданный для автоматизации скачивания видео и аудио с YouTube с обходом любых лимитов.



Проект построен вокруг современных инструментов:

\- ⚡ \*\*Telethon\*\* — асинхронное взаимодействие с Telegram API

\- 🎬 \*\*yt-dlp\*\* — сверхбыстрое получение медиаконтента

\- 🛠️ \*\*FFmpeg\*\* — умная обработка и оптимизация видео

\- 🗄️ \*\*SQLite3\*\* — хранение пользователей и статистики



Основная цель проекта — дать пользователям коммерческий загрузчик, который работает быстро, стабильно и способен обходить жесткие ограничения.



\---



\# ✨ Основные возможности



\# 🎬 YouTube Downloader

✅ Скачивание видео до \*\*1080p Full HD\*\*

✅ Извлечение оригинальных потоков YouTube

✅ Выбор качества через красивые Inline-кнопки

✅ Интеллектуальный парсинг только реально доступных форматов

✅ Конвертация видео с использованием Variable Bitrate (VBR)

✅ Извлечение MP3 Audio



\# ⚡ Умная система качества

Бот не просто скачивает первый попавшийся файл. Перед загрузкой анализируются:

\* разрешение;

\* видеокодек (H.264, VP9, AV1);

\* аудиокодек (AAC, Opus, MP3).



\# 🚀 Direct Copy Mode (Мгновенная обработка)

Если YouTube отдает поток в формате `H.264 + AAC`, бот \*\*не выполняет\*\* повторное кодирование:

`YouTube 1080p -> Direct Copy -> Telegram`

\* ⚡ Максимальная скорость (занимает 1-2 секунды)

\* 🎥 100% сохранение качества оригинала

\* 🖥️ Нулевая нагрузка на CPU

\## 🧠 Dual-Contour Architecture (Двойной контур)

Уникальная архитектура проекта, обходящая лимит ботов в 2 ГБ.



```text

&#x20;                Пользователь

&#x20;                    │

&#x20;                    ▼

&#x20;          \[ Telegram Bot Interface ]

&#x20;                    │

&#x20;       ┌────────────┴────────────┐

&#x20;       ▼                         ▼

&#x20;  \[ Bot API ]             \[ Premium Userbot ]

(Файлы до 2 ГБ)             (Файлы от 2 до 4 ГБ)

```



\* Видео до 2 ГБ прилетают от имени бота (слева).

\* Видео крупнее 2 ГБ прилетают от вашего имени (справа) прямо в то же окно!

\## 🖥️ Аппаратное ускорение (Hardware Encoding)

Поддерживаются аппаратные чипы для перекодирования тяжелых кодеков (VP9/AV1), сокращая время обработки с минут до секунд:



| Оборудование | Кодек в `.env` |

| :--- | :--- |

| \*\*NVIDIA\*\* (GTX/RTX) | `h264\_nvenc` |

| \*\*AMD\*\* (Radeon RX) | `h264\_amf` |

| \*\*Intel\*\* (QuickSync) | `h264\_qsv` |

| \*\*CPU\*\* (Универсальный) | `libx264` |

\## 👑 Административная панель (SQLite3)

Встроенная база данных с управлением через Telegram (команды доступны только владельцу):



\* `/admin` — общая статистика сервера, статус Premium и кодеков.

\* `/users` — вывести список последних 50 пользователей и их Telegram ID.

\* `/ban <ID>` — заблокировать пользователя (защита от спама).

\* `/unban <ID>` — разблокировать пользователя.

\* `/broadcast <Сообщение>` — массовая рассылка уведомлений всем юзерам.

\## 🇷🇺 Работа в РФ (Обход блокировок через Happ / Hiddify)

Для стабильной работы Telegram и YouTube в условиях блокировок, бот работает через локальный прокси-клиент \*\*Happ (Hiddify)\*\*. Обычные публичные SOCKS5-прокси не подойдут для скачивания больших объемов данных.



\*\*Пошаговая настройка:\*\*

1\. Запустите приложение Hiddify (Happ) на вашем ПК с рабочим VPN-ключом.

2\. Зайдите в настройки Hiddify и \*\*ОБЯЗАТЕЛЬНО включите галочку «Allow LAN» (Разрешить локальную сеть / Подключение через LAN)\*\*. Без этой галочки бот не сможет подключиться к туннелю!

3\. Убедитесь, что порт SOCKS5 в приложении равен `10808`.

4\. В файле `.env` вашего бота включите прокси:

&#x20;  ```env

&#x20;  USE\_PROXY=True

&#x20;  PROXY\_HOST=127.0.0.1

&#x20;  PROXY\_PORT=10808

&#x20;  ```



\*Итог: Весь трафик бота и загрузчика пойдет через ваш зашифрованный туннель на максимальной скорости.\*

\## 📥 Установка и запуск



\*\*1. Клонирование и зависимости:\*\*

```bash

git clone https://github.com/USERNAME/youtube-monster-bot.git

cd youtube-monster-bot

pip install -r requirements.txt

```



\*\*2. Настройка (`.env`):\*\*

Скопируйте `cp .env.example .env` и впишите свои данные:

```env

API\_ID=ваш\_id

API\_HASH=ваш\_hash

BOT\_TOKEN=токен\_вашего\_бота

VIDEO\_ENCODER=libx264

USE\_PROXY=True

```



\*\*3. Запуск:\*\*

```bash

python bot.py

```

Откроется интерактивное консольное меню (CLI). Нажмите `2` для включения режима \*\*Dual-Contour (4 ГБ)\*\* или `1` для стандартного запуска.
